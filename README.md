# 🌦️ Weather Card

A modern and responsive weather application that allows users to search for any city and view real-time weather information. The app is built using HTML, CSS, and JavaScript and integrates with the OpenWeather API to provide accurate weather data with a clean and user-friendly interface.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather conditions with dynamic icons
- 💧 Humidity information
- 🌬️ Wind speed display
- 📊 Atmospheric pressure
- ☀️ UV Index section
- 📅 Current date display
- ⏳ Animated loading indicator while fetching data
- ❌ Custom "City Not Found" illustration for invalid searches
- 📱 Fully responsive design for desktop and mobile devices

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- Google Material Symbols

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/palshrenik/weather-card.git
   ```

2. Navigate to the project folder:
   ```bash
   cd weather-card
   ```

3. Open `index.html` in your browser.

4. Add your own OpenWeather API key in `script.js` if needed.
